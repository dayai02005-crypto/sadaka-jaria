# موقع تذكاري - في ذمة الله

موقع إسلامي لتذكر الأحبة الذين انتقلوا إلى رحمة الله.

## المميزات

- ✨ عرض صور الأحبة
- 📖 آيات قرآنية وأدعية
- 🔍 بحث بالأسماء
- ➕ إضافة أشخاص جدد (عبر `/addsomeone`)

## تقنيات مستخدمة

- React + Vite
- CSS خالص
- API مع Vercel Serverless Functions

## النشر على Vercel

1. ادفع الكود إلى GitHub
2. استورد المشروع في Vercel
3. اضبط المتغيرات البيئية إن لزم
4. انشر!

## الإدارة

لإضافة شخص جديد، اذهب إلى `/addsomeone` واستخدم كلمة المرور المحددة في الكود.
